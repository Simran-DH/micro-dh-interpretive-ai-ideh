Micro DH and Interpretive AI for Indian Ecological Drama (IDEH/Jugaad)

This repository accompanies a research project on Micro Digital Humanities and interpretive AI methods
applied to publicly accessible plays by Girish Karnad.

What this repo contains
- Annotation schema (IDEH: Nature as Culture)
- Prompt templates (zero-shot, few-shot, metadata-grounded)
- Evaluation metrics for interpretational shift
- Metadata pack (summaries, lexicon, metaphors)
- Pilot study results tables (no copyrighted text)

 Copyright note
Due to copyright restrictions, this repository does not include full play texts or scanned pages.
All materials here are methodological, paraphrased, or metadata-based to enable reproducibility
without redistributing copyrighted content.
